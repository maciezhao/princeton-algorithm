## 💡 学习建议

### 做作业时的心态
1. **不要求完美**：第一次实现能通过测试就行
2. **边做边学**：遇到不懂的API就查文档
3. **用好AI工具**：Cursor可以帮你写boilerplate，但核心逻辑要自己理解
4. **出错是正常的**：作业就是用来练习和犯错的

### 如何高效使用Cursor
```
✅ 好的用法：
- "帮我生成Point类的基本结构"
- "这个slopeTo方法有bug，帮我检查"
- "解释一下为什么要用Double.POSITIVE_INFINITY"

❌ 不好的用法：
- "帮我写完整个作业"（失去学习机会）
- 盲目接受AI代码不思考
```

### 卡住了怎么办？
1. **先看作业指南**：`week6-collinear-points-作业指南.md`里有完整实现思路
2. **看Coursera论坛**：很多人遇到过相同问题
3. **问Cursor**：描述你的具体问题
4. **暂时跳过**：先做简单的部分，回头再攻克
